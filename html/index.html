Asymtotic Notations

What is

Asymtotic Notations

?


            Asymptotic notation is a way to describe the

performance of algorithms.

It helps us understand how the time or space requirements of an algorithm

grow as the input size increases.

O (Big O)

Big O Notation helps us understand the

 Worst case  

which the running time or space requirements of an algorithm increase as the input size grows.

O (Omega)

Omega Notation helps us understand the

 Best case  

which describes the lower limit of the time complexity.

O (Theta)

Theta Notation helps us understand the

 Average case  

which describes the exact time complexity.

It tells us that the time taken by an algorithm is tightly bound between an upper and a lower limit.
